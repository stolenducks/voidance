# Voidance Desktop Environment Keyboard Shortcuts Reference

## Quick Reference

### Essential Shortcuts
- `Super + Return` - Open terminal
- `Super + D` - Application launcher
- `Super + R` - Run command
- `Super + Q` - Close window
- `Super + L` - Lock screen

### Navigation
- `Super + Arrow Keys` - Navigate windows
- `Super + 1-9` - Switch workspaces
- `Super + Shift + Arrow Keys` - Move windows

## Tips & Tricks

### Productivity Tips

1. **Workspace Organization**
   - Use workspaces 1-3 for active work
   - Reserve workspace 9 for communication apps
   - Use workspace 8 for system monitoring

2. **Window Management**
   - Use `Super + Shift + Arrow Keys` to organize windows quickly
   - `Super + Space` helps navigate between many windows in a column
   - `Super + F` for temporary fullscreen viewing

3. **Application Launching**
   - Type partial names in wofi for fuzzy matching
   - Use `Super + R` for quick command execution
   - Press `Tab` in wofi to expand search

### Navigation Shortcuts

1. **Quick Movement**
   - `Super + Home/End` for column navigation
   - `Super + Ctrl + Home/End` for window navigation
   - Arrow keys for precise movement

2. **Layout Switching**
   - `Super + S` for spiral (good for coding)
   - `Super + T` for stack (good for reference)
   - `Super + Y` for horizontal (good for comparison)

### System Management

1. **Session Control**
   - Use `session-manager menu` for full session control
   - `Super + L` for quick screen locking
   - `Super + Ctrl + R` after config changes

2. **Hardware Control**
   - Media keys work out of the box
   - Volume and brightness keys are configured
   - Screenshot tools are readily available

## Customization

### Adding Custom Shortcuts

Edit `~/.config/niri/config.kdl` to add custom shortcuts:

```kdl
// Example custom shortcuts
bind Mod+Ctrl+B { spawn "firefox" }
bind Mod+Ctrl+E { spawn "thunar" }
bind Mod+Ctrl+T { spawn "telegram-desktop" }
```

### Application-Specific Shortcuts

Some applications have their own shortcuts that may conflict:

- **Terminal**: `Ctrl+Shift+C/V` for copy/paste
- **Browser**: `Ctrl+T` for new tab, `Ctrl+W` to close tab
- **File Manager**: `Ctrl+N` for new window, `F5` to refresh

### Learning Path

1. **Week 1**: Master essential shortcuts (terminal, launcher, navigation)
2. **Week 2**: Learn workspace and layout management
3. **Week 3**: Practice advanced window manipulation
4. **Week 4**: Customize with personal shortcuts

## Troubleshooting

### Shortcuts Not Working

1. Check if another application is using the shortcut
2. Verify Niri configuration syntax: `niri msg validate-config`
3. Restart Niri: `Super + Ctrl + R`

### Common Conflicts

- **Terminal**: May intercept `Ctrl+C` for interrupt
- **Browser**: May use `Ctrl+T` for new tabs
- **IDE**: May have custom keybindings

### Reset to Defaults

If shortcuts become corrupted:

```bash
# Backup current config
cp ~/.config/niri/config.kdl ~/.config/niri/config.kdl.backup

# Reset to defaults
cp /etc/skel/.config/niri/config.kdl ~/.config/niri/config.kdl

# Reload configuration
Super + Ctrl + R
```
