# Keyboard Shortcuts

## Essential Shortcuts

| Shortcut | Action |
|----------|--------|
| `Super + Return` | Open terminal (Alacritty) |
| `Alt + D` | Open app launcher (Walker) |
| `Super + Q` | Close focused window |
| `Super + M` | Exit Hyprland |
| `Super + F` | Toggle fullscreen |
| `Super + Space` | Toggle floating mode |

---

## Window Navigation

| Shortcut | Action |
|----------|--------|
| `Super + ←/→/↑/↓` | Move focus (arrow keys) |
| `Super + H/J/K/L` | Move focus (vim keys) |
| `Super + 1-9` | Switch to workspace 1-9 |
| `Super + Mouse wheel` | Cycle through workspaces |

---

## Window Management

| Shortcut | Action |
|----------|--------|
| `Super + Shift + ←/→/↑/↓` | Move window |
| `Super + Shift + 1-9` | Move window to workspace 1-9 |
| `Super + Mouse drag` | Move window (floating) |
| `Super + Right-click drag` | Resize window |

---

## System

| Shortcut | Action |
|----------|--------|
| `Super + L` | Lock screen |
| `Super + Shift + R` | Reload Hyprland config |
| `Print` | Screenshot (full) |
| `Shift + Print` | Screenshot (area) |

---

## Applications

| Shortcut | Action |
|----------|--------|
| `Super + E` | File manager (Thunar) |
| `Super + B` | Browser (Firefox) |
| `Super + C` | ChatGPT web app |
| `Super + N` | Notion web app |

---

## Audio

| Shortcut | Action |
|----------|--------|
| `XF86AudioRaiseVolume` | Volume up |
| `XF86AudioLowerVolume` | Volume down |
| `XF86AudioMute` | Mute/unmute |
| `XF86AudioPlay` | Play/pause media |

---

## Customization

Edit these keybindings in:
```bash
nano ~/.config/hypr/hyprland.conf
# or
nano config/hypr/hyprland.conf  # (in project repo)
```

After editing, reload with `Super + Shift + R`
